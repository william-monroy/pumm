Introducción a CSS - Parte 2

¿De qué trata la sesión?

CSS es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado. En esta sesión vimos como manipular el box model, además aprendimos a como incluir y cambiar colores de distintas formas. Finalmente vimos como trabajar con la cuadrícula, elemento muy importante para dar la organización deseada a nuestras páginas.
