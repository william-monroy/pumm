Introducción a CSS - Parte 1

¿De qué trata la sesión?

CSS es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado. Es muy usado para establecer el diseño visual de los documentos web e interfaces de usuarios escritas en HTMl. Se ha convertido en una tecnología usada por muchos sitios web para crear páginas visualmente atractivas, interfasces de usuario para aplicaciones web y GUIs para muchas aplicaciones móviles.
